# دليل إضافة تقييم جديد

## الخطوات لإضافة تقييم جديد:

### 1. تسجيل الدخول بالحساب الصحيح

**مهم:** يجب أن تسجل دخول بحساب له `entity_id` (جهة مرتبطة):

✅ **يمكنه إضافة تقييمات:**
- Ministry Admin: `admin@modt.gov` / `password123`
- Entity User: `user@ncc.gov` / `password123`

❌ **لا يمكنه إضافة تقييمات:**
- Super Admin: `admin@system.gov` (هذا الحساب للإدارة فقط، ليس له جهة)

### 2. الوصول إلى صفحة إنشاء التقييم

بعد تسجيل الدخول بالحساب الصحيح، لديك 3 طرق:

#### الطريقة الأولى - من لوحة التحكم:
1. اذهب إلى: http://localhost:5173/dashboard
2. ستجد زر **"+ تقييم جديد"** في أعلى اليمين
3. اضغط عليه

#### الطريقة الثانية - من صفحة التقييمات:
1. اذهب إلى: http://localhost:5173/assessments
2. ستجد زر **"+ تقييم جديد"** في أعلى اليمين
3. اضغط عليه

#### الطريقة الثالثة - مباشرة:
اذهب مباشرة إلى: http://localhost:5173/assessment/new

### 3. ملء الاستمارة (6 خطوات)

ستظهر لك الاستمارة بـ 6 خطوات:

1. **الخطوة 1:** المعلومات العامة
   - إجمالي الموظفين
   - موظفو تقنية المعلومات
   - موظفو الأمن السيبراني

2. **الخطوة 2:** البنية التحتية
   - هل لديكم بنية تحتية؟
   - (إذا نعم: تظهر حقول إضافية)

3. **الخطوة 3:** الخدمات الرقمية
   - أنواع الخدمات
   - معلومات الموقع والبريد

4. **الخطوة 4:** الأمن السيبراني (مهم!)
   - معايير الامتثال
   - أدوات الأمان

5. **الخطوة 5:** المراقبة والموافقات
   - الموافقات الأمنية
   - تكرار التقارير

6. **الخطوة 6:** التقنيات المتقدمة
   - المحاكاة الافتراضية
   - VPN/APIs
   - اختبارات الأمان

### 4. الحفظ والتقديم

- كل خطوة تُحفظ تلقائياً عند الضغط على **"التالي"**
- يمكنك الخروج والعودة لاحقاً (حالة: مسودة)
- بعد إتمام جميع الخطوات، اضغط **"تقديم التقييم ✓"**

---

## إذا لم يظهر زر "تقييم جديد"

### السبب المحتمل:
أنت مسجل دخول بحساب **Super Admin** الذي ليس له جهة مرتبطة.

### الحل:
1. اضغط **"تسجيل الخروج"** من أعلى اليمين
2. سجل دخول بأحد هذه الحسابات:
   - **Ministry Admin:** `admin@modt.gov` / `password123`
   - **Entity User:** `user@ncc.gov` / `password123`
3. الآن سيظهر الزر
