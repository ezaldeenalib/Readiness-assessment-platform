## 📱 الصفحات الجديدة

افتح هذه الروابط بعد تسجيل الدخول:

### 🏦 بنك الأسئلة
**الرابط:** `/question-bank`

**ماذا ستجد:**
- قائمة الأسئلة مع فلاتر
- زر "+ إضافة سؤال جديد"
- 5 أنواع من الأسئلة
- قواعد تقييم مخصصة

**ماذا تفعل:**
1. انقر "+ إضافة سؤال جديد"
2. اختر النوع (مربوط بالبيانات / يدوي / اختيار متعدد / نعم-لا)
3. املأ النص والإعدادات
4. احفظ

---

### 🎯 القوالب
**الرابط:** `/templates`

**ماذا ستجد:**
- بطاقات القوالب (3 قوالب جاهزة)
- عدد الأسئلة والنقاط
- أزرار (عرض، تعديل، نسخ، حذف)

**لإنشاء قالب جديد:**
1. انقر "إنشاء قالب جديد"
2. أدخل الاسم والوصف
3. انقر "+ إضافة سؤال"
4. اختر من بنك الأسئلة
5. رتب وحدد الأقسام
6. احفظ

---

### 📋 التقييمات
**الرابط:** `/template-assessments`

**ماذا ستجد:**
- قائمة التقييمات مع النتائج
- زر "إنشاء تقييم جديد"
- فلاتر (جهة، قالب، حالة)

**لإنشاء تقييم:**
1. انقر "إنشاء تقييم جديد"
2. اختر الجهة والقالب
3. **✨ التعبئة التلقائية تحدث!**
4. أكمل الأسئلة المتبقية
5. احفظ وأرسل

---

### 📊 البيانات الثابتة
**الرابط:** `/static-data`

**ماذا ستجد:**
- نموذج بيانات الجهة
- حقول مجمعة حسب الفئات
- معلومات عن النسخ التاريخية

**ماذا تفعل:**
1. اختر جهة
2. املأ البيانات
3. احفظ
4. **✨ نسخة تاريخية جديدة تُنشأ!**

## 🌟 المميزات الرئيسية

### 1. التعبئة التلقائية ✨
```
السؤال: "هل لدى المنظمة أكثر من 500 موظف؟"
النوع: مربوط بالبيانات
الحقل: total_employees

عند إنشاء التقييم:
→ النظام يقرأ total_employees تلقائياً
→ يعبئ الإجابة: 650
→ يقيّم: 650 > 500 ✅
→ النتيجة: 10/10 نقاط
→ المستخدم يراها قراءة فقط 🔒
```

### 2. التصحيح التلقائي ⚙️
```
9 قواعد تقييم:
> أكبر من
< أصغر من
= يساوي
≠ لا يساوي
≥ أكبر أو يساوي
≤ أصغر أو يساوي
⊃ يحتوي على
∃ موجود
⊆ ضمن نطاق

النتيجة تُحسب تلقائياً بناءً على القواعد
```

### 3. النسخ التاريخية 📅
```
عند تحديث البيانات الثابتة:
  القديم: is_active = false ✗
  الجديد: is_active = true ✓

التقييمات تحتفظ بـ snapshot:
  → لا تتأثر بالتحديثات اللاحقة
  → دقة تاريخية 100%
```

### 4. القوالب المرنة 🎨
```
سؤال واحد في قوالب متعددة
تخصيص الوزن لكل قالب
تنظيم في أقسام
إعادة ترتيب سهلة
```
